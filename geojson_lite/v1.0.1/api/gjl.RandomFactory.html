Class: RandomFactory

gjl.RandomFactory

GeoJSON Lite's random GeoJSON generator. After instantiation, you can create random geometries, features, and feature collections with the object's methods.

Constructor

new RandomFactory(opt_options)

Parameters:
Name Type Description
opt_options object Object of parameters.
Properties
Name Type Attributes Default Description
type string <optional>
 "Point" Geometry type. Supported types are Point, Line, and Polygon.
vertices integer <optional>
 5 Number of vertices (ignored for points).
stride integer <optional>
 2 Number of dimensions.
bbox array <optional>
 [-180,-90,180,90] Bounding box of the geometries. Must contain stride*2 numbers.
numProp integer <optional>
 0 Number of properties to create features with.
propSample object | undefined <optional>
 A sample of properties. Must be a plain object with numProp members.

Example
var generator = new gjl.RandomFactory({
	type: "Point",
	stride: 3
});
var randomPoint = generator.generateGeometry();
var randomPointFeature = generator.generateFeature();
var tenRandomPointsInAFeatureCollection = generator.generateFeatures(10);

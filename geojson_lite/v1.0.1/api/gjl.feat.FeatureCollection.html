Class: FeatureCollection

gjl.feat.FeatureCollection

Base class for creating GeoJSON FeatureCollection objects from an array of features. They can be gjl.feat.Feature objects, or regular GeoJSON objects.

Constructor

new FeatureCollection(feats)

Parameters:
Name Type Description
feats array Array of feature objects.
Source:
  • gjl.js, line 398
Example
var featCol = new gjl.feat.FeatureCollection([
	new gjl.feat.Feature(new gjl.geom.Point([0,0]), {id: 0, name: "myPoint"}),
	{
		type: "Feature",
		geometry: {
			type: "Point",
			coordinates: [5,5]
		},
		properties: {
			id: 1,
			name: "mySecondPoint"
		}
	}
]);

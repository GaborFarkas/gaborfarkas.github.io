Thesis examples

This page contains all of the examples for the study "Comparison of Web Mapping Libraries for Building WebGIS Clients".

It is advised to use a web server for benchmarking, as Benchmark.js's access to the browser's high resolution clock is not guaranteed when opened from the file system.

When accessing from the filesystem with Google Chrome, CORS should be enabled. CORS can be enabled by starting Chrome with the --allow-file-access-from-files flag. Note that this method causes a potential security risk. To avoid this, take care not to load any other .html files other than the ones included in this appendix in the same browsing session.
